### Project Information

**1. [V7614 PyCalc (V7614-PyCalc)](https://github.com/V7614Tech/V7614-PyCalc)** (Python)
A console-based calculator built in Python, now discontinued.
My first project that was published here on GitHub! It used to be here on this profile before I moved it. It's code is a mess, so it's currently going through a rewrite as VCalc or V7614 Calc.

**2. [WinKill (WinKill)](https://github.com/V7614Tech/WinKill)** (Batch & PowerShell)
This is a tool which is basically a shortcut to the Force shutdown command in Windows if the computer won't shut down properly from the start menu or anywhere else normally. Can also be used to restart Explorer.

**3. [Pokemon Merah & Biru (pokered-IDN)](https://github.com/PITG/pokered-IDN)** (Game Boy Assembly)
An Indonesian Translation of the original worldwide releases, Pokemon Red and Blue based on [pret/pokered](https://github.com/pret/pokered). It's mostly faithful with translation, and is a work in progress.

**4. [Pokemon Merah & Biru + (pokered-IDN-Plus)](https://github.com/V7614/pokered-IDN-Plus)** (Game Boy Assembly)
Currently a placeholder, but will be a fork of the base [PITG/pokered-IDN](https://github.com/PITG/pokered-IDN-Plus) but with added Quality of Life and Bug Fixes.

**5. [Pokemon Red & Blue 2001 (pokered-2001)](https://github.com/V7614/pokered-2001)**
Currently a placeholder. Is a fork aiming to make a "What if? Pokemon Red & Blue was released in 2001" merging [dannye/pokered-gbc](https://github.com/dannye/pokered-gbc), [dannye/pokered-crysaudio](https://github.com/dannye/pokered-crysaudio), & [dannye/pokered-gen-2-gfx](https://github.com/dannye/pokered-gen-2-gfx), along with a reorchestration of the Gen 1 music to use the full potential of Crystal's sound engine while being more faithful than the Gen 2 Kanto Soudtracks.
